json - текстовый формат передачи данных, самое распространенное применение: обмен данными между сервером и браузером. Легче воспринимается читающим, по сравнению с XML. 
xml - позволяет структурировать информацию разного типа, универсальный формат обмена данными, например, между частями программы
yaml - универсальный язык сериализации данных. Язык разметки.
